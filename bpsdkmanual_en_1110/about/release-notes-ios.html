4.13.0

Added Supported Printers

  • TD-4425DN
  • TD-4525DN
  • TD-4455DNWB
  • TD-4555DNWB
  • TD-4425DNF
  • TD-4555DNWBF

  • RJ-3255WB

  • RJ-4255WB
  • PT-E720BT
  • PT-E920BT

New Features

  • Added the new API and related class for authentication.
    • See BRLMChannelCredential.
  • Added new methods into BRLMPrinterDriverGenerator.
    • See openChannel:withCredential:.
  • Added new methods into BRLMPrinterDriver.
    • See transferPrinterConfigurationFiles:progress:.
  • Added new methods into BRPtouchPrinter for authentication support.
    • See setCredential:.

API Enhancements or Changes

  • Delete the description of the method printImageWithClosures:settings: in BRLMPrinterDriver .
  • Enhanced documentation with additional notes for CutMark functionality in BRLMPTPrintSettings regarding behavior on models that do not support CutMark.
