BRPtouchLabelInfoStatus

BRPtouchLabelInfoStatus has properties for a label set in your printer. This is for PT and QL series.

Properties

labelID

Indicate a label size set in your printer. See also LabelIdType

Declaration

@property (nonatomic) LabelIdType labelID;

labelColor

Indicate a label color set in your printer. See also ColorType

Declaration

@property (nonatomic) ColorType labelColor;

fontColor

Indicate a font color set in your printer. See also ColorType

Declaration

@property (nonatomic) ColorType fontColor;
