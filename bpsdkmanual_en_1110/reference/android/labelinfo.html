LabelInfo

com.brother.ptouch.sdk.LabelInfo class stores information about the paper loaded in the printer that is connected.

Properties

labelNameIndex

Label index number.

Declaration

int labelNameIndex;

labelColor

Label background color.

Declaration

LabelColor labelColor;

labelFontColor

Label text color specifically for PT series.

Declaration

LabelColor labelFontColor;
