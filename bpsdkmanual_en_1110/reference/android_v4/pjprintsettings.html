PJPrintSettings

Settings to print on PJ series.

Overview

You use this object to set parameters to your printer such as a paper size. The images or PDFs you passed to print functions in PrinterDriver are converted to printable data for specified printer by using this settings. Therefore, you receive an error if the paper size you set to this object don't correspond with an actual paper size set in your printer.

This class also implements PrintImageSettings, allowing you to configure image-specific print settings.
